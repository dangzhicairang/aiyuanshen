## Gacha Theme Asset Slots

Put your legal open-source assets under `Assets/Resources/GachaTheme/` with these names:

- `panel_bg.png` -> main panel background
- `btn_single.png` -> single draw button
- `btn_ten.png` -> ten draw button
- `btn_skip.png` -> skip button
- `card_bg.png` -> default result card background

Recommended free sources:

- [Kenney UI Pack](https://kenney.nl/assets/ui-pack-rpg-expansion)
- [OpenGameArt UI](https://opengameart.org/art-search-advanced?keys=ui+fantasy)
- [Pixabay SFX](https://pixabay.com/sound-effects/search/magic/)

After placing files, return to Unity and wait for import. The scene auto-applies them at runtime.
