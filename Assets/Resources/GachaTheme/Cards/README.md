# Card Portrait Dropzone

Place runtime card portraits here. The loader expects `Resources` paths from `gacha_card_catalog.json`.

## Expected file names

- `c_001.png`
- `c_002.png`
- `c_101.png`
- `c_102.png`
- `c_103.png`
- `w_501.png`
- `w_401.png`
- `w_402.png`
- `w_301.png`
- `w_302.png`
- `w_303.png`
- `w_304.png`

After import, Unity path example:

- File: `Assets/Resources/GachaTheme/Cards/c_001.png`
- Runtime load path: `GachaTheme/Cards/c_001`
